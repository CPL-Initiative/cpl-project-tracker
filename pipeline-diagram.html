How It Works
⏰ Daily at 6 AM PT
GitHub Actions runs automatically on a schedule. Fetches fresh CCCCO live metrics and Custom Report data, runs the Python pipeline, and pushes directly to main. GitHub Pages deploys in seconds. No manual steps needed.
Fully automatic
↻ On-Demand Refresh
The Refresh Live Data button in the dashboard header triggers the same GitHub Actions workflow immediately via the Cloudflare Worker. Use it any time you need a mid-day update without waiting for the 6 AM schedule.
Manual trigger
🎯 One Pipeline
The Cowork Routine was retired — the Anthropic cloud sandbox cannot reach the CCCCO Azure API or Cloudflare Worker. GitHub Actions is now the single, reliable pipeline for both scheduled and on-demand updates.
